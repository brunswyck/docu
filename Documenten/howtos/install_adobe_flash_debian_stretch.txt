Download the latest release of the plugin in tar.gz format from https://get.adobe.com/flashplayer/
As root, extract the tar.gz package and copy libflashplayer.so to /usr/lib/flashplugin-nonfree
Give the file sane permission/ownership values: 


guru@vdab10:~/Downloads$ sudo cp libflashplayer.so /usr/lib/flashplugin-nonfree/
[sudo] password for guru: 
guru@vdab10:~/Downloads$ sudo chmod 644 /usr/lib/flashplugin-nonfree/libflashplayer.so 
guru@vdab10:~/Downloads$ sudo chown root:root /usr/lib/flashplugin-nonfree/libflashplayer.so 



guru@vdab10:~/Downloads$ sudo update-alternatives --list flash-mozilla.so
